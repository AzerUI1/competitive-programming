## Explanation

The problem is asking us to determine how many dollars the soldier needs to borrow in order to buy `w` bananas, given that the cost of the `i`-th banana is `i * k` dollars. We can calculate the total cost of the bananas using the formula for the sum of the first `w` natural numbers, which is `w * (w + 1) / 2`. Then we can multiply this sum by `k` to get the total cost. Finally, we can subtract the amount of money `n` that the soldier has from the total cost to find out how much he needs to borrow. If the result is negative, it means he doesn't need to borrow any money.

## Implementation
**Time Complexity:** $O(1)$

```cpp
#include <iostream>

using namespace std;

int main() {
    int k, w, n;
    cin >> k >> w >> n;

    long long total_cost = (long long)w * (w + 1) / 2 * k;
    long long borrow_amount = total_cost - n;

    if (borrow_amount < 0) {
        borrow_amount = 0;
    }

    cout << borrow_amount << endl;
    return 0;
}

```
