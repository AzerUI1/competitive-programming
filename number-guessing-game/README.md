##  About The Game

The Number Guessing Game is a simple yet entertaining game where players try to guess a randomly generated number between 0 and 100. With only 5 attempts, every guess counts! Whether you're a beginner learning Python or just looking for a quick mental challenge, this game is perfect for you.

##  Features

-  **Random Number Generation**: Every game is unique with numbers between 0-100
-  **5 Attempts Only**: Makes the game challenging and exciting
-  **Real-time Feedback**: Get hints whether to guess higher or lower
-  **Replay Option**: Play as many times as you want
-  **Colorful Console Output**: Enhanced visual experience with emojis and formatting
-  **Error Handling**: Prevents crashes from invalid inputs

##  How to Play

1. Run the game
2. The computer will randomly select a number between 0 and 100
3. You have **5 attempts** to guess the correct number
4. After each guess, you'll receive a hint:
   -  "Try a bigger number!" - if your guess is too low
   -  "Try a smaller number!" - if your guess is too high
5. Guess correctly within 5 attempts to win!
