## Explanation

For each code $i$ we are given a vector $A_i$ of length $M$.
We also have a vector $B$ of length $M$ and a constant $C$.

The condition for code $i$ to be accepted is:

$$
\sum_{j=1}^{M} A_{ij} \cdot B_j + C > 0.
$$

So we can just compute this value for every code and count how many are positive.

## Implementation

**Time Complexity:** $O(N \cdot M)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	int n, m;
	long long c;
	cin >> n >> m >> c;

	vector<long long> b(m);
	for (int j = 0; j < m; j++) {
		cin >> b[j];
	}

	int accepted_count = 0;
	for (int i = 0; i < n; i++) {
		long long score = c;
		for (int j = 0; j < m; j++) {
			long long a_ij;
			cin >> a_ij;
			score += a_ij * b[j];
		}
		if (score > 0) {
			accepted_count++;
		}
	}

	cout << accepted_count << '\n';
	return 0;
}
```
