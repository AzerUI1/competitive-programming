## Explanation

We process participants from rank `1` to rank `N`.

Keep two counters:

- `passed`: total number of accepted participants so far.
- `overseas_passed`: number of accepted overseas students (`'b'`) so far.

Rules while scanning each character in `S`:

- If `S[i] == 'a'` (Japanese student):
  Accept if `passed < A + B`.
- If `S[i] == 'b'` (overseas student):
  Accept if both are true:
  - `passed < A + B`
	- `overseas_passed < B`
- If `S[i] == 'c'` (not a student):
  Reject.

For each participant, print:

- `Yes` if accepted
- `No` otherwise

## Implementation

**Time Complexity:** $O(N)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	int n, a, b;
	string s;
	cin >> n >> a >> b;
	cin >> s;

	int passed = 0;
	int overseas_passed = 0;

	for (int i = 0; i < n; i++) {
		if (s[i] == 'a') {
			if (passed < a + b) {
				cout << "Yes" << '\n';
				passed++;
			} else {
				cout << "No" << '\n';
			}
		} else if (s[i] == 'b') {
			if (passed < a + b && overseas_passed < b) {
				cout << "Yes" << '\n';
				passed++;
				overseas_passed++;
			} else {
				cout << "No" << '\n';
			}
		} else {
			cout << "No" << '\n';
		}
	}

	return 0;
}
```
