## Explanation

We are given four digits $A,B,C,D$ and must choose operators `+` or `-` between them so that:

$$A \;op_1\; B \;op_2\; C \;op_3\; D = 7.$$

There are only $2^3 = 8$ ways to choose `(op1, op2, op3)`, so we can brute force all combinations.

For each combination, evaluate the expression, and once it equals `7`, print it in the required format (no spaces) followed by `=7`.

The statement guarantees at least one solution.

## Implementation

**Time Complexity:** $O(1)$ (8 checks)

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	string s;
	cin >> s;

	int a = s[0] - '0';
	int b = s[1] - '0';
	int c = s[2] - '0';
	int d = s[3] - '0';

	for (int mask = 0; mask < 8; mask++) {
		int value = a;
		char op_1 = (mask & 1) ? '+' : '-';
		char op_2 = (mask & 2) ? '+' : '-';
		char op_3 = (mask & 4) ? '+' : '-';

		value += (op_1 == '+') ? b : -b;
		value += (op_2 == '+') ? c : -c;
		value += (op_3 == '+') ? d : -d;

		if (value == 7) {
			cout << a << op_1 << b << op_2 << c << op_3 << d << "=7\n";
			return 0;
		}
	}

	return 0; // guaranteed not to happen
}
```
