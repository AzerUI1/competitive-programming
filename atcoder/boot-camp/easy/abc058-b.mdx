## Explanation

We are given two strings:

- `O`: characters at odd positions
- `E`: characters at even positions

To restore the original password, we simply merge them one by one:

1. Take the first character from `O`.
2. Take the first character from `E`.
3. Take the second character from `O`.
4. Take the second character from `E`.
5. Continue until both strings are fully used.

If `O` is longer by one character, its last character is appended at the end automatically by the loop.

## Implementation

**Time Complexity:** $O(|O| + |E|)$

```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
    string o, e;
    cin >> o >> e;

    string ans;
    for (int i = 0; i < (int)o.size(); i++) {
        ans += o[i];
        if (i < (int)e.size()) {
            ans += e[i];
        }
    }

    cout << ans << '\n';

    return 0;
}
```
