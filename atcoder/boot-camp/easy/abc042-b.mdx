## Explanation

All strings have the same length `L`. To get the lexicographically smallest concatenation, we should place smaller strings earlier.

So the strategy is:

1. Read all `N` strings into an array.
2. Sort the array in ascending lexicographic order.
3. Concatenate strings in that order.
4. Print the result.

Because every string has equal length, normal lexicographic sorting directly gives the optimal order.

## Implementation

**Time Complexity:** $O(N \log N \cdot L)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	int n, l;
	cin >> n >> l;

	vector<string> s(n);
	for (int i = 0; i < n; i++) {
		cin >> s[i];
	}

	sort(s.begin(), s.end());

	string ans;
	for (const string& x : s) {
		ans += x;
	}

	cout << ans << '\n';
	return 0;
}
```
